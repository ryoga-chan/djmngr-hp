Ready-made importable datasets
Create an import folder and download a dataset into it then follow instructions based on your preferred installation type .

Available datasets:

  • doujinshi.org @ 2020-10 / 9.2MiB (~50MiB imported)
Import file using docker
Add to the docker command the following parameter: 
-v /path/to/import_folder:/app/import
then start the app and wait some minutes for the task to finish.
Import file using docker-compose
In the volumes section of compose.yml add the following line: 
- /path/to/import_folder:/app/import
then start the app and wait some minutes for the task to finish.
Import file using the local installation
Move the import folder into the app root directory, then run the command: 
./bin/rails djorg:metadata:load RAILS_ENV=production
wait some minutes for the task to finish.
