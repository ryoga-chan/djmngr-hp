Simpler interface for smaller/low-power devices
A simpler browsing interface with basic HTML and Javascript for old tablets/phones or ereaders is available by clicking the top-right tablet icon or by visiting http://localhost:PORT/?format=ereader (where port is 39102 for the local installation and 3000 for docker).
An on-screen virtual keyboard is also available in case of need by pressing the KB button beside a text input field.
